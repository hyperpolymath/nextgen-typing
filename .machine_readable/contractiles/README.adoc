= Contractiles
:toc:
:sectnums:

This directory contains the flat contractile set for nextgen-typing
(estate-standardization layout, adopted 2026-06-12). The six contractile
verbs are: intend, must, trust, adjust, bust, dust.

== Contents

* `Intentfile.a2ml` — north-star: purpose, anti-purpose, committed
  next-actions, horizon wishes (verb: intend; non-gating)
* `Mustfile.a2ml` — hard physical-state invariants; CI fails on violation
  (verb: must)
* `Trustfile.a2ml` — agent trust levels/boundaries + integrity invariants
  (verb: trust)
* `Adjustfile.a2ml` — drift tolerances + accessibility invariants
  (verb: adjust; advisory)
* `Bustfile.a2ml` — failure modes, recovery paths, escalation ladder
  (verb: bust)
* `Dustfile.a2ml` — cleanup/hygiene items (verb: dust; advisory)
* `Justfile` — contractile runner recipes

== History

Until 2026-06-12 the contractiles lived as nested directories
(`must/`, `trust/`, `dust/`, `bust/`) plus root-level `*.contractile`
files (MUST/TRUST/ADJUST/INTENT). The estate-standardization merge
flattened the layout; the old files' real content was ported into the
flat files above (see each file's provenance header). The retired `lust`
verb was folded into `Intentfile` back in April 2026 (commit 9634e38).
